# X To Everything

将内容生成为 X (Twitter) 风格推文截图，支持多页、图片混排。

## 功能

- 文本与图片按顺序混排
- 按高度自动分页（支持长图切割）
- 9:16 / 3:4 比例
- 白天 / 黑夜主题
- 中英文展示
- 单页 / 全部下载 PNG

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

项目已配置 Vercel 部署，推送至 GitHub 后可在 [Vercel](https://vercel.com) 导入仓库自动部署。
